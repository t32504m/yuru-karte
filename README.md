# **🌿 ゆるカルテ**

「このお客様、前回どこを重点的に施術したっけ…？」をなくすための、個人サロン向けシンプル顧客管理アプリ。

## **概要**

「ゆるカルテ」は、個人事業主のマッサージ師や整体師、リラクゼーションサロン運営者が、顧客の基本情報・施術履歴・次回予約をスマートフォンでサッと記録し、確認するためのWebアプリケーションです。

「よっ友メモ」をフォークして開発されており、軽快な動作と使いやすさはそのままに、サロン業務に特化した機能とデザインを取り入れています。

インストールや面倒な会員登録は一切不要。ブラウザから index.html を開くだけですぐに使用できます。

## **特徴**

* **予約前に「思い出す」ことに特化**: 本格的なCRMではなく、施術前に「誰か」「前回どうだったか」「次回どうするか」をすばやく振り返るためのUI設計。  
* **直感的なカルテ機能**: 施術の好み（強さなど）や注意事項、重点部位をタグや短いメモで管理。  
* **履歴と予約のシームレスな連携**: 顧客ごとに施術履歴と次回予約を紐付けて保存。  
* **プライバシーに配慮した設計**: 外部サーバーやクラウドへの送信は一切なし。データはすべて端末内に保存されます。  
* **モバイルファースト＆手帳デザイン**: 40代の女性施術者でも使いやすい、温かく清潔感のある「ダスティピンク」を基調としたやわらかいUI。

## **データ保存について**

このアプリはブラウザの内部データベース（IndexedDB）を使用してデータを保存します。

* 外部データベースへの送信やクラウド同期を行わないため、お客様のプライバシーが守られます。  
* 同じ端末・同じブラウザを使用している限りデータは保持されますが、**ブラウザのサイトデータやキャッシュをクリアするとデータが消失する可能性があります**。

### **データ管理（バックアップ・移行）**

「データ管理」メニューから以下の操作が可能です。定期的なバックアップを強く推奨します。

* **エクスポート**: 顧客データ、履歴、予約データを1つのJSONファイルとして端末にダウンロード保存できます。  
* **インポート**: 保存したJSONファイルを読み込み、データの復元や別端末への引き継ぎが可能です（現在のデータにマージされます）。

## **使い方**

1. このリポジトリをクローンまたはZIPでダウンロードします。  
2. index.html を任意のスマートフォンやPCのブラウザで開きます。  
3. iPhone(Safari)やAndroid(Chrome)の「ホーム画面に追加」機能を使用すると、ネイティブアプリのように全画面で快適に利用できます。  
4. 右上の「＋ 顧客追加」ボタンから、お客様情報の登録を始めてください。

## **技術スタック**

* **Frontend**: HTML5, JavaScript (Vanilla JS)  
* **Styling**: Tailwind CSS (CDN)  
* **Database**: IndexedDB (Vanilla JS API)
